    Uncertainty Quantification (F2-01)

    This guide covers MC Dropout for H2, uncertainty propagation through the non-negative logistic stacker, and temperature scaling.

    Quickstart

    1) H2 MC samples (N,S) using your H2 model:

    from chiss.heads.h2_uncertainty import predict_with_mc_dropout
unc = predict_with_mc_dropout(h2_model, X_tensor, n_samples=50)
np.save("artifacts/stage2/h2_mc.npy", unc.samples)  # (N,S)

2.  Propagate through stacker:

make uncertainty-predict H2_MC=artifacts/stage2/h2_mc.npy \
  STACKER_W=artifacts/stage2/stacker_weights.json \
  IDS=artifacts/stage2/ids.csv \
  OUT=artifacts/stage2/pred_with_uncertainty.csv

    3.  Calibrate probabilities (temperature scaling):

make uncertainty-calibrate CSV=artifacts/stage2/oof_with_probs.csv
cat artifacts/uncertainty/calibration.json

Output Columns
    •   p_mean, p_std, p_lower, p_upper where lower/upper are 5%/95% quantiles by default.

Notes
    •   MC Dropout enables stochasticity during inference by forcing dropout layers to train mode while keeping batch norm (if present) frozen.
    •   H1 epistemic uncertainty is derived from fold variance; H3 treated as deterministic unless provided as samples.

## Pipeline Integration (F2-02)

1) **Inference with uncertainty + calibration**
```bash
make infer-uncertain IDS=artifacts/stage2/ids.csv \
  STACKER_W=artifacts/stage2/stacker_weights.json \
  H2_MC=artifacts/stage2/h2_mc.npy \
  OUT=artifacts/stage2/pred_with_uncertainty.csv \
  ARGS="--h1-folds artifacts/stage2/h1_fold_0.npy artifacts/stage2/h1_fold_1.npy --calibration artifacts/uncertainty/calibration.json"

    Output CSV columns: star,p_mean,p_std,p_lower_90,p_upper_90,score_risk_adjusted.

    2) Build calibration report & plots

    make uncertainty-report CSV=artifacts/stage2/oof_with_probs.csv

    Artifacts: - artifacts/uncertainty/report/reliability.png - artifacts/uncertainty/report/sigma_vs_rmse.png - artifacts/uncertainty/report/sigma_hist.png - artifacts/uncertainty/report/report.json

    3) CI Gates - Gate R²(σ, RMSE) ≥ 0.90 - ECE and Brier must not worsen after temperature scaling.

    Confidence-aware ranking

    We compute score_risk_adjusted = p_mean / sqrt(p_std^2 + eps). Use this for alerting and priority queues; retain raw p_mean for science decisions.

    Fallback Calibration (F2-04)

    Isotonic (non-parametric) calibration

    make uncertainty-calibrate-iso CSV=artifacts/stage2/oof_with_probs.csv

    Creates artifacts/uncertainty/calibration_isotonic.json with piecewise-constant mapping.

    Split-conformal interval calibration

    make uncertainty-calibrate-intervals CSV=artifacts/stage2/oof_with_probs.csv

    Adds interval parameters (bin-conditional on p_std) to calibration_full.json.

    Coverage Gate: Report includes interval_coverage vs target (default 90%) and enforces ±3% tolerance.

    Ranking & Dossier Surfacing (F2-03)

    Rank with Pareto & risk score

    make rank-candidates IN=artifacts/stage2/pred_with_uncertainty.csv \
  OUT=artifacts/stage2/ranked.csv \
  ARGS="--min-prob 0.5 --max-sigma 0.20 --topk 100"

    This produces ranked.csv with rank,score_risk_adjusted,pareto_layer.

    Inject uncertainty badges into dossiers

    make dossiers-uncertainty PRED_CSV=artifacts/stage2/ranked.csv \
  DOSSIERS_DIR=artifacts/long_period

    Each dossier.html gains a header badge showing p_mean ± σ, 90% CI, and risk score.
